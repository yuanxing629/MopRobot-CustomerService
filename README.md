# Aster扫地机器人智能客服系统

基于大语言模型和RAG技术的智能客服解决方案，为扫地机器人用户提供精准的产品咨询和个性化的使用报告服务。

## 功能特性

- **智能问答**：基于RAG技术，结合知识库为用户提供准确的产品咨询答复
- **个性化报告**：集成外部用户数据，生成包含使用特征、清洁效率、耗材情况的个人报告
- **场景感知**：动态提示词切换机制，自动识别普通咨询与报告生成场景
- **流式响应**：支持实时流式输出，提升用户体验
- **多格式文档支持**：支持PDF、TXT格式的知识库文档自动加载与向量化

## 核心模块

| 模块 | 说明 |
|------|------|
| `agent/` | ReAct智能体核心，集成工具链与中间件系统 |
| `rag/` | 向量存储与检索服务，支持MD5去重机制 |
| `model/` | LLM与嵌入模型工厂，基于阿里云DashScope |
| `utils/` | 配置管理、日志处理、文件加载等工具类 |

## 快速开始

### 环境要求

- Python 3.11+
- 阿里云DashScope API Key

### 安装依赖

```bash
pip install -r requirements.txt
```

### 配置

在 `config/` 目录下编辑配置文件：

- `rag.yml`：配置模型参数
- `chroma.yml`：配置向量数据库
- `agent.yml`：配置外部数据路径

### 运行

```bash
streamlit run app.py
```

## 项目亮点

1. **动态提示词切换**：通过上下文感知中间件，实现普通咨询与报告生成场景的智能切换
2. **完整监控体系**：工具调用全流程日志记录与性能监控
3. **企业级设计**：工厂模式、配置化管理、完善的错误处理机制

## 技术栈

- **LLM框架**：LangChain / LangGraph
- **向量数据库**：ChromaDB
- **嵌入模型**：阿里云 text-embedding-v4
- **大语言模型**：阿里云 qwen3-max
- **前端框架**：Streamlit
